**Basic reinforcement learning package**

**Algorithms:**
- DP (policy & value iteration)
- Monte Carlo (on-policy & off-policy)
- TD (SARSA, q-learning, expected SARSA, double q-learning)

**Examples:**
- K-bandits
- Car rentals
- Gambler
- Racetrack

**Notes:**
- All on-policy algorithms use epsilon-greedy
